# Toy React

A toy React, developed in TypeScript

- fiber
- FunctionComponent
- useEffect
- useState
- 其他的 hook 有空添加


## Usage

**install dependencies**

```shell
yarn install
```

**development**

```shell
yarn dev
```

**production**

```shell
yarn build
```
